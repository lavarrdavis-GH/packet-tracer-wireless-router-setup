# Packet Tracer: Configure a Wireless Router and Clients

## Project Overview
This project demonstrates the configuration of a home network using Cisco Packet Tracer. The network includes a cable modem, a home wireless router, two PCs, a laptop, and a TV. The steps include connecting devices, configuring the wireless router, setting up DHCP, and testing network connectivity.

## Objectives
- Connect network devices and cables correctly
- Configure the wireless router GUI for secure home networking
- Set up DHCP to assign IP addresses to hosts
- Configure a wireless LAN and secure it with WPA2
- Test connectivity from wired and wireless clients

## File Structure
- `PacketTracerFiles/`: Contains the Packet Tracer `.pkt` file for this project
- `Instructions/`: Step-by-step guides for each part of the project
- `Images/`: Screenshots of GUI configurations and device status
- `Notes/`: Network diagrams and supporting documentation

## Usage
1. Open `WirelessRouterSetup.pkt` in Cisco Packet Tracer.
2. Follow the instructions in the `Instructions/` folder.
3. Test connectivity by accessing `skillsforall.srv` from all devices.
